# Voron-trident-config
This repository contains my Voron Trident configuration. As a starting point, I installed an XoL Afterburner with a CAN cable to simplify the connection between the toolhead and the mainboard, a Manta M8P V2.0. The toolhead uses an EBB36 V1.2, and the hotend is a Dragon High Flow.

## Main features
- Voron Trident 300
- Manta M8P V2.0 + CB1
- EBB36 CAN
- Dragon HF
- Sherpa Mini
- XoL
- Tap sensor
